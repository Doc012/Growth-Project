# MIE Java OCA Study Group: The Zero-to-Hero Calculator Project

Welcome to our collaborative Java project. We are students at Mecer Inter Ed, working together to master the concepts for the Java OCA certification.

### The Mission
We are building a calculator, but not just any calculator. We're starting small to get the fundamentals right, with a clear plan to evolve it into a powerful scientific calculator that can handle real-world calculations. This project is our hands-on lab for every topic we cover in our OCA study guide. As a scalable project, today it's just a calculator... only God knows what's gonna be tomorrow!

### The Plan (How We'll Grow)
This project is designed to grow in complexity, mirroring our own learning journey:
1.  **Phase 1: The Simple Calculator (We are here!)**
    - Focus: Java basics, methods, `if`/`switch` statements, primitive data types.
    - Goal: A command-line program that performs basic arithmetic (+, -, *, /).
2.  **Phase 2: The Loop-Driven Calculator**
    - Focus: `while` and `do-while` loops, `break`, `continue`.
    - Goal: Allow the user to perform multiple calculations without restarting the program.
3.  **Phase 3: The Scientific Calculator**
    - Focus: Arrays, `StringBuilder`, Core APIs (`Math` class, `ArrayList`, Wrapper classes).
    - Goal: Add features like power, square root, trigonometric functions, and a history of calculations.
4.  **Phase 4: The Object-Oriented Calculator**
    - Focus: Classes, Objects, Constructors, Encapsulation, Inheritance, Polymorphism.
    - Goal: Refactor the code into a well-structured, extensible system (e.g., an `Operation` interface with classes like `Addition`, `Subtraction`).
5.  **Phase 5: The Robust Calculator**
    - Focus: Exception Handling.
    - Goal: Make the program bulletproof by gracefully handling errors like division by zero or invalid input.

Let's build this, learn together, and ace the OCA!
